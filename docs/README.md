# Статичний сайт CyberSimulator для GitHub Pages

Цей проект було перетворено на статичний веб-сайт для розміщення на GitHub Pages.

## 🌐 Доступ до сайту

Після публікації на GitHub Pages, сайт буде доступний за адресою:
```
https://[ваш-username].github.io/[назва-репозиторію]/
```

## 📋 Що було змінено

### До (Flask додаток):
- Серверна частина на Python (Flask)
- База даних SQLite
- Серверна автентифікація
- Потребує хостингу з підтримкою Python

### Після (Статичний сайт):
- Тільки HTML, CSS, JavaScript
- LocalStorage для зберігання даних
- Клієнтська автентифікація
- Працює на GitHub Pages безкоштовно

## 🚀 Налаштування GitHub Pages

1. Перейдіть у налаштування репозиторію: Settings → Pages
2. Оберіть джерело: Deploy from a branch
3. Виберіть гілку: `main` або `master`
4. Виберіть папку: `/docs`
5. Натисніть Save

## 💾 Зберігання даних

Дані зберігаються в LocalStorage браузера:
- `users` - список користувачів
- `currentUserId` - ID поточного користувача
- `testLogs` - результати тестів

**Важливо:** Дані зберігаються локально у браузері кожного користувача окремо.

## 🔧 Локальне тестування

Щоб протестувати локально, відкрийте `docs/index.html` у браузері або використайте локальний сервер:

```bash
# Python 3
cd docs
python -m http.server 8000

# Або через Node.js
npx serve docs
```

Потім відкрийте: http://localhost:8000
